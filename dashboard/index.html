Connect your sheet

The dashboard reads your job applications directly from Google Sheets. You need two things: your Sheet ID and a Sheets API key.

Finding your Sheet ID:
It's in the URL: docs.google.com/spreadsheets/d/THIS_PART/edit

Getting an API key:
console.cloud.google.com → APIs & Services → Credentials → Create API key. Enable the Google Sheets API. Restrict the key to Sheets API only.

These values are saved in your browser only. Never sent anywhere else.
